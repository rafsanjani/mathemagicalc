Algorithm for converting any decimal number to a binary number

This algorithm is valid for any binary number (n) where n can either be a whole number or a fraction

Given a decimal number(N), we differentiate the number into fractional and whole parts
The binary equivalent of the whole part is computed using the steps in converting a decimal fraction to binary
The binary equivalent of the fractional part is computed using the steps in converting a decimal integer to binary
The results obtained in step 3 is joined to the results obtained in step 2 to produce the binary equivalent of the decimal number provided
