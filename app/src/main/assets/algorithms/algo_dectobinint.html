Algorithm for converting a decimal whole number to a binary number

This algorithm is valid only for a number (n) where n is an integer greater than or equal to one

Given a decimal number(N), let No be the number to convert. That is, let No = N
If No is an odd number, take bo = 1
If No is an even number, take bo = 0
Using the information gathered in step 1 and step 2, the binary number can be computed using the iterative equation
$$N_{k+1}=\frac{N_{k-b_{k}}}{2}$$
where bk is taken as 1 if Nk is odd or 0 if Nk is even.
The value obtained is reused in subsequent iterations until the stopping criteria is met i.e Nk becomes 0
The final binary equivalent of the decimal number is read in terms of bk in the order bkbk-1bk-2...b0
where k is the number of iterations performed in step 3
