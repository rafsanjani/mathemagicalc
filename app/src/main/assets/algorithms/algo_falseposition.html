Algorithm for approximating the root of an equation using the False Position Method

This algorithm is valid for equations of the form f(x) = 0

Given that the root of a function f(x) lies in a given interval $[x_{0}, x_{1}]$

Compute $f_{0} = f(x_{0})$ and $f_{1} = f(x_{1})$;
compute $$x^{1} = x_{2} = x_{1} - \left ( \frac{x_{1}-x_{0}}{f_{1}-f_{0}} \right )f_{1}$$
Compute $f_{2} = f(x_{2})$
Case 1
If $f(x_{0})f(x_{2})<0$, then the root lies in $[x_{0}, x_{2}]$.
Discard the interval $[x_{2}, x_{1}]$ and take $[x_{0}, x_{2}]$ as new interval by setting $x_{1} = x_{2}$

Case 2
If $f(x_{2})f(x_{1}) < 0$, then the root lies in $[x_{2}, x_{1}]$.
Discard the interval $[x_{0}, x_{1}]$ and take $[x_{2}, x_{1}]$ as new interval by setting $x_{0} = x_{2}$
if $$\left | \frac{x_{0}-x_{1}}{x_{0}} \right | < tol$$ then $x^{*} = x_{2}$
STOP:
Else go back to step 2
